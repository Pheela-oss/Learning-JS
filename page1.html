O que é Arrays?

Imagine Arrays como uma bandeja: Essa bandeja tem um nome específico e dentro dessa bandeja você "cospe" números, letras, todo tipo de valor que pode ser armazenado, por exemplo: 


				const dados: {
					nome: Kuromi
					idade: XX
					signo: escorpião
				}

Ele é muito usado e muito importante para guardar valores que vão ser usados mais cedo ou mais tarde, tais como dados de usuário, items de um jogo, ou até mesmo um inventário de jogos, tal como o exemplo: 


				const inventario: {
					martelo,
					espada,
					picareta,
					poção
				}

				console.log(inventario[2])
				// resultado: espada
